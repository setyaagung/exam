## Ujian Online

Bahasa Pemrograman : PHP dan Laravel

Aplikasi Ujian Online ini sangat berguna bagi sekolah supaya memudahkan guru memberikan tugas maupun ujian secara online kepada siswa serta memudahkan laporan untuk nilai siswa
tanpa harus mengoreksi satu per satu.

Fitur :

-   CRUD Kelas
-   CRUD Mata Pelajaran
-   CRUD User
-   CRUD Ujian
-   Memiliki 3 hak akses yaitu admin, guru dan siswa
-   Soal ujian berupa pilihan ganda
